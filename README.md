# MCP Manager

A powerful Web-based GUI tool for managing Model Context Protocol (MCP) servers in Claude and Cursor. It supports multiple transport protocols and provides a unified MCP proxy service, allowing AI clients to access all enabled tools by configuring just one server.

## üöÄ Core Features

### üåê **Complete Multi-Transport Protocol Support**
**Now fully supports four transport methods, thoroughly tested and verified. AI clients can stably access your MCP services through any protocol!**

- **üì° SSE (Server-Sent Events)**: `GET /sse` - Server-side real-time message push, supports unidirectional communication
- **üîÑ HTTP JSON-RPC**: `POST /mcp` - Standard MCP protocol interface, fully compatible with MCP specifications, supports RESTful calls
- **‚ö° WebSocket**: `ws://host:port/ws` - Bidirectional real-time communication with low latency, supports full-duplex communication
- **üñ•Ô∏è Traditional stdio**: Command-line method - Original standard input/output method, backward compatible

### üéõÔ∏è **Unified Proxy Service Architecture**
**Revolutionary single-point access design - No need to configure multiple MCP servers in AI clients, just configure one mcp-manager to access all tools!**

### üîß **Complete Graphical Management System**
**Easily manage all MCP servers through an intuitive web interface, real-time monitoring of connection status**

### ‚úÖ **Verified Stability**
**All transport layer functions have been thoroughly tested to ensure stable operation in production environments**

### ‚ö° **Method 2: WebSocket (High Performance‚ú®)**
**Bidirectional real-time communication, suitable for scenarios requiring frequent interaction. Verified to work perfectly!**

```
// JavaScript client example
const ws = new WebSocket('ws://127.0.0.1:3456/ws');

ws.onopen = () => {
  console.log('‚úÖ WebSocket connection established');
  
  // Send initialization request
  ws.send(JSON.stringify({
    "jsonrpc": "2.0",
    "id": 1,
    "method": "initialize",
    "params": {
      "protocolVersion": "2024-11-05",
      "capabilities": {"tools": {}},
      "clientInfo": {"name": "websocket-client", "version": "1.0.0"}
    }
  }));
};

ws.onmessage = (event) => {
  const response = JSON.parse(event.data);
  console.log('üì• Received response:', response);
  
  // Handle initialization response, request tools list
  if (response.id === 1 && response.result) {
    ws.send(JSON.stringify({
      "jsonrpc": "2.0",
      "id": 2,
      "method": "tools/list",
      "params": {}
    }));
  }
};

ws.onerror = (error) => {
  console.error('‚ùå WebSocket error:', error);
};

ws.onclose = () => {
  console.log('üîå WebSocket connection closed');
};
```

### üì° **Method 3: SSE (Real-time Push‚ú®)**
**Server-sent event stream, suitable for scenarios requiring real-time monitoring of updates. Verified to work perfectly!**

```
// Establish SSE connection
const eventSource = new EventSource('http://127.0.0.1:3456/sse');

eventSource.onopen = () => {
  console.log('‚úÖ SSE connection established');
};

eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('üì• Received server message:', data);
  
  if (data.type === 'connected') {
    console.log('‚úÖ SSE connection confirmed, client ID:', data.clientId);
  }
};

eventSource.onerror = (error) => {
  console.error('‚ùå SSE error:', error);
};

// Send JSON-RPC request via SSE endpoint
fetch('http://127.0.0.1:3456/sse/rpc', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/list",
    "params": {}
  })
}).then(response => response.json())
  .then(data => console.log('üöÄ Tools list:', data));
```

### üîå **Complete WebSocket Testing Example**
**Verified working WebSocket test code:**

```
// Save as test-websocket.js and run: node test-websocket.js
import WebSocket from 'ws';

const ws = new WebSocket('ws://localhost:3456/ws');

ws.on('open', function open() {
    console.log('‚úÖ WebSocket connection established');
    
    const initRequest = {
        "jsonrpc": "2.0",
        "id": 1,
        "method": "initialize",
        "params": {
            "protocolVersion": "2024-11-05",
            "capabilities": {"tools": {}},
            "clientInfo": {"name": "websocket-test-client", "version": "1.0.0"}
        }
    };
    
    console.log('üì§ Sending initialization request...');
    ws.send(JSON.stringify(initRequest));
});

ws.on('message', function message(data) {
    const response = JSON.parse(data.toString());
    console.log('üì• Received message:', response);
    
    if (response.id === 1 && response.result) {
        console.log('‚úÖ Initialization successful, requesting tools list...');
        const toolsRequest = {
            "jsonrpc": "2.0",
            "id": 2,
            "method": "tools/list",
            "params": {}
        };
        ws.send(JSON.stringify(toolsRequest));
    } else if (response.id === 2) {
        console.log('‚úÖ Tools list retrieval complete');
        console.log('üîß Available tools count:', response.result?.tools?.length || 0);
        ws.close();
    }
});

ws.on('close', function close() {
    console.log('üîå WebSocket connection closed');
    process.exit(0);
});
```
